Commomn formatting Tags

  1. b tag
    • This tag is used to apply bold formatting
  2. i tag
    • This tag is used to apply italic formatting text
  3. u tag
    • This tag is used to underline the text
  4. em Tag
    • This tag is used to emphasize the text,typically rendered in italics
  5. br tag
    • This tag is used to create a line break
  6. sup tag
    • This tag is used to apply superscript of formatting to text
  7. sub tag
    • This tag is used to apply subscript of formatting to text
