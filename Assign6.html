HTML Tags

Tags are used to define elements in an HTML document.

  • Tags are surrounded by angle brackets (< >) and are used to mark up content on a web page.
  • HTML tags can be either opening tags or closing tags, and they define the beginning and end.

1)Heading Tags

Heading tags is used to mark headings in HTML
From H1 to H6 We have tags for the most important to the least important heading

2)Paragraph Tag

The (p) tag defines a paragraph.
The Paragraph tag is used to add paragraph in to an HTML page

3)Anchor tag

The anchor tag is is used to add links to an existing content inside an HTML page

4)Image Tag

Img tag is used to add image in an HTML page

5)Bold,Italic and Underline Tags

We can use bold ,italic and underline tags to highlight the text as follows

  • Bold tag is used to bold the text
  • Italic tag is used to apply italic style to the text
  • Underline tag is used to underline the text
